        The 10-Day Journey
        
One day at a time, from a blank screen to a live website

        
Each day builds directly on the last — nothing is taught that isn't used.

      


      


        

          
01

          
Introduction to Websites and Artificial Intelligence
Understanding how the internet works, the role of web developers, and introducing AI as a tool for assistance and productivity.

        


        

          
02

          
HTML Fundamentals
Learn the building blocks of the web—structuring content with headings, paragraphs, links, list items, and images.

        


        

          
03

          
CSS and Website Design
Styling elements, applying color palettes, selecting modern typography, and understanding page spacing to design clean interfaces.

        


        

          
04

          
Bootstrap and Responsive Design
Using the Bootstrap framework to build mobile-friendly grid layouts that look great on any screen size.

        


        

          
05

          
JavaScript Fundamentals
Adding interactivity to pages, handling events, and creating simple features like mobile navigation toggle menus.

        


        

          
06

          
Learning to Use AI Tools Responsibly
Exploring how to leverage AI tools for learning, planning, and debugging while preserving core logic and understanding.

        


        

          
07

          
Build a Restaurant Website
A complete hands-on project to design a multi-section restaurant page with styled menus, image galleries, and contact details.

        


        

          
08

          
Build a Hotel Website
Building a booking landing page with room displays, modern layouts, amenities, and reservation forms.

        


        

          
09

          
Publish Projects Online using GitHub
Learning Git/GitHub basics to deploy and host student projects live online with accessible shareable URLs.

        


        

          
10

          

            
Project Showcase, Career Guidance and Certificate Distribution

            
Students present their live portfolios, receive career guidance for tech fields, and get their Certificate of Completion.

            

              
              Supabase Pro Challenge: Students who finish early will receive an optional advanced challenge: an introduction to Login & Signup systems using Supabase Authentication.

School Infrastructure Requirements

            
To ensure a smooth bootcamp experience, the school is requested to provide the following simple resources:

            
    
              
  • 
                
                
    Computer Lab or Classroom: Access to computer laboratory systems (students are also encouraged to bring their own laptops for an optimal experience).
    
              
    • 
              
  • 
                
                
    Internet Connection: Standard network internet (preferred) to upload code and publish websites live.
    
              
    • 
              
  • 
                
                
    Visual Projector: A projector or display screen (if available) for the instructor's live coding demonstrations.
    
              
    • 
              
  • 
                
                
    School Coordinator: One dedicated staff coordinator to assist with scheduling and communication.
    
              
    • 
            

          

        


        

          

            
Participation & Program Guidelines

            
Standard guidelines for conducting the summer program at the school campus:

            
    
              
  • 
                
                
    Orientation Session: A brief interactive orientation session is held for students and parents before classes begin to preview projects and answer queries.
    
              
    • 
              
  • 
                
                
    Minimum Enrollment: The program commences with a minimum enrollment of 5 students.
    
              
    • 
              
  • 
                
                
    Mini Hackathon: A special, friendly hackathon is organized on the final day if the total enrollment reaches 15 or more students.
    
              
    • 
              
  • 
                
                
    School-Branded Certificate: Every student receives a Certificate of Completion featuring the school name, logo, date, and signatures of the Principal and the Instructor.
